# Sales Performance Analysis Dashboard (Excel)

## Project Overview
This project is an interactive Excel dashboard created to analyze sales performance across products, regions, and months.  
It helps identify key business insights such as top-performing products, revenue trends, and profitability.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Excel Formulas

## Key Features
- KPI cards for Total Revenue, Total Units Sold, Total Profit
- Product-wise and Region-wise revenue analysis
- Monthly sales trend visualization
- Revenue vs Profit comparison
- Interactive slicers for Product, Region, and Month

## Dataset
The dataset is a simulated sales dataset designed to reflect real-world business scenarios.

## Key Insights
- Laptops generate the highest revenue and profit
- North region contributes the maximum revenue
- Sales show noticeable variation across months
- High revenue does not always imply high profit

## Learning Outcomes
- Hands-on experience with Excel pivot tables and dashboards
- Business-oriented data analysis
- Creating interactive and decision-friendly reports
